How HungerWatch Works

How It Works

Each time someone visits a "Find Food" page on participating food bank websites, HungerWatch quietly records a single, anonymous event. Over time, these small signals form an early warning system for changing food demand patterns.

What We Collect

  • Domain name of the food bank website
  • Timestamp of the page visit
  • That's it. No personal data, cookies, or tracking.

For Food Banks: Adding the Monitor

Participating in HungerWatch is simple. Just add this small JavaScript snippet to your "Find Food" or food assistance pages: 

<script>
/*
  HungerWatch – anonymous pageview signal
  Add this once, preferably at the bottom of your Find Food page.
  It sends a tiny, no-cookie ping to the HungerWatch endpoint.
*/

(function() {
  const endpoint = "https://kvytpubwfaszoserxjmr.supabase.co/functions/v1/counter";
  try {
    fetch(endpoint, { keepalive: true });
  } catch (e) {
    // Fail silently – never affect the page
  }
})();
</script>

Technical Note: This script runs silently and will never interfere with your website's functionality. If the HungerWatch service is unavailable, it fails gracefully without affecting your page.
